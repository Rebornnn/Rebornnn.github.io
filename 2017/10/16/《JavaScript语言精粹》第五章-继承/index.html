函数化(Functional)

继承模式的一个弱点是没法保护隐私，函数化模式可以解决这个问题。简单分四步：

  1. 创建一个新对象
  2. 有选择的定义私有实例变量和方法。函数中通过var定义的普通变量。
  3. 给这个对象扩充方法。这些方法有特权去访问参数，以及第2步中通过var定义的变量。
  4. 返回那个新对象。
