CSS规范简介、作者编码原则

CSS规范

CSS规范由CSS工作组的成员编写，CSS工作组成员组成：

  • W3C 会员公司的成员（88%）
  • 特邀专家（7%）
  • W3C 工作人员（5%）

每项规范从最初启动到最终成熟，都会经过以下阶段。

  1. 编辑草案（ED）
  2. 首个公开工作草案（FPWD）
  3. 工作草案（WD）
  4. 候选推荐规范（CR）
  5. 提名推荐规范（PR）
  6. 正式推荐规范（REC）

CSS 是被打散到多个不同的规范（模块）中，每个模块都可以独立更新版本。这其中，那些延续 CSS 2.1 已有特性的模块会升级到3这个版本号。
此外，如果某个模块是前所未有的新概念，那它的版本号将从 1 开始。

最常见的前缀分别是：

  • Firefox 的 -moz-
  • IE 的 -ms-
  • Opera的 -o-
  • Safari 和 Chrome 的-webkit-
